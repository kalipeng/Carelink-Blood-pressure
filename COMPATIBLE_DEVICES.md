# 🩺 Compatible Blood Pressure Monitors

## ✅ 我们的系统可以支持的设备

只要是**支持蓝牙（Bluetooth/BLE）**的血压计，理论上都可以使用！

---

## 🔵 蓝牙血压计的两种协议

### 1️⃣ 标准蓝牙协议 (推荐)

**特征：**
- 使用 BLE Blood Pressure Service (UUID: 0x1810)
- 符合医疗设备标准
- 不需要专用 SDK
- **我们的代码可以直接支持** ✅

**常见品牌：**
- ✅ OMRON (欧姆龙) - 大部分型号
- ✅ A&D Medical
- ✅ Beurer
- ✅ Withings
- ✅ QardioArm
- ✅ 大部分医疗级设备

### 2️⃣ 私有协议

**特征：**
- 使用厂商自定义协议
- 需要专用 SDK 或 App
- 需要调试数据格式
- **我们的代码需要修改** ⚠️

**常见品牌：**
- ⚠️ 一些便宜的品牌
- ⚠️ 一些较老的型号

## 🔍 如何确认设备有蓝牙

### 方法1：查看产品名称
```
✅ 型号包含 "BT" 或 "Bluetooth"
✅ 产品描述写 "Wireless" 或 "Bluetooth Enabled"
✅ 包装盒上有蓝牙标志 🔵
```

### 方法2：查看功能
```
✅ 提到可以连接 App
✅ 提到数据同步
✅ 写明 "Bluetooth Smart" 或 "BLE"
```

### 方法3：查看型号
```
OMRON 系列：
- ✅ BP786, BP791IT, BP761 → 有蓝牙
- ❌ BP710, BP710N → 无蓝牙
```
