# 📱 iHealth KN-550BT 血压计激活指南

## 🚨 问题：iPhone 设置里看不到这个设备

**这是最常见的问题！** 意味着血压计没有正确开机或进入配对模式。

---

## 🔧 完整激活步骤（5 分钟）

### Step 1：检查电池（1 分钟）

#### 打开电池仓
```
1. 翻转血压计到背面
2. 找到电池仓盖
3. 按照箭头方向滑开或打开
```

#### 检查电池
```
[ ] 电池已安装
[ ] 电池方向正确（+ - 极性对应）
[ ] 电池有电（建议使用新电池）
[ ] 电池接触良好（不松动）
```

**iHealth KN-550BT 通常使用：**
- 4 节 AA 电池（5 号电池）或
- 1 个专用电池包

**如果电池没电或快没电：**
```
❌ 设备无法启动蓝牙
❌ iPhone 无法发现设备
❌ 连接会不稳定
```

---

### Step 2：正确开机（2 分钟）

#### 方法 A：标准开机（最常见）

```
1. 找到设备正面的 "电源" 或 "START" 按钮
2. 短按一次（1 秒）
3. 观察屏幕是否亮起
4. 等待 2-3 秒让设备完全启动
```

**成功的标志：**
- ✅ 屏幕显示 "0" 或 "--"
- ✅ 可能显示蓝牙图标 📶
- ✅ 可能听到"滴"一声
- ✅ LED 指示灯可能闪烁

---

#### 方法 B：长按开机（如果方法 A 无效）

```
1. 长按 "电源" 或 "START" 按钮 3-5 秒
2. 观察屏幕反应
3. 等待设备完全启动
```

---

#### 方法 C：强制开机（如果设备卡住）

```
1. 取出所有电池
2. 等待 30 秒
3. 重新装入电池
4. 短按电源按钮
```

---

### Step 3：进入蓝牙配对模式（关键！）⭐⭐⭐⭐⭐

**这是最关键的一步！** 很多蓝牙设备开机后需要手动进入配对模式。

#### 🎯 常见方法（按顺序尝试）

---

#### 方法 1：自动配对模式（最简单）

某些 iHealth 设备开机后会自动进入配对模式：

```
1. 按电源按钮开机
2. 等待 5-10 秒
3. 观察是否有蓝牙图标闪烁
4. 此时查看 iPhone 蓝牙设置
```

**成功的标志：**
- ✅ 蓝牙图标闪烁（不是常亮）
- ✅ 屏幕显示 "bt" 或 "Bluetooth" 字样
- ✅ LED 灯快速闪烁（某些型号）

---

#### 方法 2：长按进入配对模式（推荐）⭐⭐⭐⭐⭐

**这是最可能的方法：**

```
1. 确保设备已开机（屏幕亮着）
2. 找到 "SET" 或 "MEMORY" 或 "USER" 按钮
3. 长按这个按钮 5-10 秒
4. 观察屏幕或指示灯变化
```

**可能的按钮组合（尝试这些）：**

##### 组合 A：单按钮长按
```
长按 "MEMORY" 按钮 5 秒
或
长按 "SET" 按钮 5 秒
或
长按 "电源" 按钮 5 秒
```

##### 组合 B：双按钮同时按
```
同时按住 "电源" + "MEMORY" 按钮 3-5 秒
或
同时按住 "START" + "SET" 按钮 3-5 秒
```

##### 组合 C：连续按键
```
连续快速按 "MEMORY" 按钮 3 次
然后
长按 "SET" 按钮 5 秒
```

**成功进入配对模式的标志：**
- ✅ 屏幕显示 "Pair" 或 "bt" 或 "BLE"
- ✅ 蓝牙图标开始快速闪烁（每秒 2-3 次）
- ✅ LED 灯快速闪烁
- ✅ 设备发出"滴滴"声
- ✅ 屏幕可能显示 "---" 或 "rdy"（ready）

---

#### 方法 3：重置设备（终极方法）

如果上述方法都不行，尝试恢复出厂设置：

```
1. 确保设备开机
2. 取出袖带（如果已安装）
3. 找到设备背面或侧面的 "RESET" 孔
4. 用牙签或回形针插入 RESET 孔
5. 按住 3-5 秒
6. 观察屏幕是否重置（显示 "0.0" 或闪烁）
```

**或者尝试软件重置：**
```
1. 同时按住所有按钮 10 秒
2. 等待设备完全关机
3. 再次开机
```

**重置后：**
- ✅ 设备会清除所有配对记录
- ✅ 设备会自动进入配对模式
- ✅ 这时应该能在 iPhone 上看到了

---

### Step 4：在 iPhone 上验证（1 分钟）

#### 打开 iPhone 蓝牙设置

```
1. 打开 iPhone "设置" app
2. 点击 "蓝牙"
3. 确保蓝牙开关是绿色（已开启）
4. 等待 5-10 秒让 iPhone 扫描
```

#### 查看设备列表

在 "其他设备" 部分查找：

**可能的设备名称：**
- ✅ "KN-550BT"
- ✅ "iHealth KN-550BT"
- ✅ "iHealth BP"
- ✅ "BP Monitor"
- ✅ "Blood Pressure"
- ✅ "KN-550"
- ✅ "iHealth" + 一串数字
- ✅ 设备 MAC 地址（如 "A1:B2:C3:..."）

**如果看到类似的名称：**
```
🎉 恭喜！设备已进入配对模式
➡️ 继续下一步
```

**如果还是看不到：**
```
⚠️ 继续往下看故障排查部分
```

## 🔍 详细故障排查

### 问题 1：屏幕不亮 / 设备不开机

**可能原因：**
- 电池没电
- 电池装反了
- 电池接触不良
- 设备损坏

**解决方法：**

#### A. 检查电池
```
1. 取出电池
2. 检查电池极性（+ - 对应）
3. 用其他设备测试电池是否有电
4. 确认电池型号正确（AA 或设备专用）
5. 尝试更换新电池
```

#### B. 清洁电池接触点
```
1. 用干布或橡皮擦清洁电池仓内的金属接触点
2. 清洁电池两端
3. 重新安装电池
4. 按紧电池确保接触良好
```

#### C. 检查设备
```
1. 检查是否有物理损坏
2. 检查电池仓弹簧是否正常
3. 如果仍然不开机，可能是设备故障
```

---

### 问题 2：屏幕亮了但 iPhone 看不到设备

**这是最常见的问题！** 说明设备开机了但没有进入配对模式。

**解决方法（按顺序）：**

#### Step A：确认蓝牙功能
```
1. 查看设备屏幕是否有蓝牙图标 📶
2. 如果没有，说明蓝牙功能未激活
3. 尝试 "Step 3 方法 2" 中的按钮组合
```

#### Step B：重启设备
```
1. 关闭设备（长按电源键或取出电池）
2. 等待 10 秒
3. 重新开机
4. 立即查看 iPhone 蓝牙列表（在开机后 10 秒内）
```

**为什么？** 某些设备只在开机后的短时间内可被发现。

#### Step C：缩短距离
```
1. 将血压计和 iPhone 放在一起（< 10 cm）
2. 移除所有金属物体
3. 远离其他电子设备
4. 重新扫描
```

#### Step D：强制刷新 iPhone 蓝牙
```
1. 在 iPhone 设置中关闭蓝牙
2. 等待 5 秒
3. 重新打开蓝牙
4. 等待自动扫描
```

#### Step E：重启 iPhone
```
1. 完全关机 iPhone
2. 等待 10 秒
3. 重新开机
4. 立即打开蓝牙设置
5. 重新开启血压计
```

---

### 问题 3：之前配对过，现在找不到了

**可能原因：**
- 设备已经在配对列表中
- 连接状态异常
- 需要重新配对

**解决方法：**

#### A. 检查 "我的设备" 列表
```
1. 打开 iPhone 设置 > 蓝牙
2. 查看 "我的设备" 部分（不是 "其他设备"）
3. 查找 "KN-550BT" 或类似名称
```

**如果找到了：**
```
情况 1：显示 "已连接"
   → 太好了！设备已连接，直接使用 app

情况 2：显示 "未连接"
   → 点击设备名称尝试连接
   → 如果连接失败，继续下一步

情况 3：显示 "连接中..." 一直不变
   → 点击设备名称右边的 (i)
   → 点击 "忽略此设备"
   → 确认忽略
   → 重新配对
```

#### B. 忽略设备后重新配对
```
1. 点击设备名称右边的 (i) 图标
2. 点击 "忽略此设备"
3. 确认忽略
4. 关闭血压计
5. 等待 10 秒
6. 重新开启血压计并进入配对模式
7. 在 "其他设备" 中重新扫描
```

---

### 问题 4：看到了设备但连接失败

**可能原因：**
- 设备同时连接到其他设备
- 配对码不匹配
- 蓝牙协议不兼容

**解决方法：**

#### A. 断开其他连接
```
1. 如果血压计之前连接过其他手机/平板
2. 在那个设备上断开连接
3. 或者重置血压计（清除所有配对）
```

#### B. 检查配对码
```
某些设备需要配对码（PIN）：
- 常见配对码：0000, 1234, 1111
- 查看设备说明书或包装盒
- 某些设备配对码显示在屏幕上
```

#### C. 使用 app 配对（而不是系统蓝牙）
```
‼️ 重要：BLE 设备通常不应该在系统蓝牙中配对！

正确方法：
1. 不要在 iPhone 设置 > 蓝牙 中点击设备
2. 直接运行你的 carelink app
3. 让 app 自动扫描并连接
4. BLE 设备不需要在系统中"配对"
```

## 💡 关键提示

### 🎯 最重要的 3 点：

1. **确保电池有电** - 低电量会导致蓝牙无法启动
2. **进入配对模式** - 仅仅开机是不够的，必须进入配对模式
3. **不要在系统蓝牙中配对** - BLE 设备应该通过 app 连接

### ⚠️ 常见误区：

```
❌ 错误：在 iPhone 设置 > 蓝牙 中点击设备配对
✅ 正确：只确认设备出现，然后用 app 连接

❌ 错误：设备开机后立即尝试连接
✅ 正确：等待 5-10 秒让设备完全启动

❌ 错误：设备在房间另一边
✅ 正确：设备应该在 < 1 米距离内

❌ 错误：使用快没电的旧电池
✅ 正确：使用全新的或刚充满电的电池
```
