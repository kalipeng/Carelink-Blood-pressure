# 📱 一页搞定：KN-550BT 连接指南

## 🎯 目标：让 iPhone 能看到血压计

---

## ⚡ 3 步快速解决（5 分钟）

### Step 1：准备血压计（2 分钟）

```
1. 装入新电池（4 节 AA 电池）
2. 按 "电源" 按钮开机
3. 等待屏幕亮起
```

**成功标志：** 屏幕显示 "0" 或 "--"

---

### Step 2：进入配对模式（2 分钟）⭐⭐⭐⭐⭐

**🚨 最关键的一步！**

#### 方法 A（推荐）：
```
1. 设备开机后
2. 长按 "MEMORY" 按钮 5-10 秒
3. 观察蓝牙图标是否开始闪烁
```

#### 方法 B（备用）：
```
1. 关闭血压计
2. 取出电池等待 30 秒
3. 重新装入电池
4. 按电源开机
```

**成功标志：**
- ✅ 蓝牙图标快速闪烁（每秒 2-3 次）
- ✅ 屏幕可能显示 "Pair" 或 "bt"

---

### Step 3：验证 iPhone 能看到（1 分钟）

```
1. 打开 iPhone "设置" app
2. 点击 "蓝牙"
3. 确保蓝牙开关是绿色
4. 在 "其他设备" 中查找：
   • "KN-550BT" 或
   • "iHealth KN-550BT" 或
   • "iHealth BP"
```

**✅ 看到设备了？**
```
太好了！
❌ 不要点击设备！
✅ 直接运行 carelink app
```

**❌ 还是看不到？**
```
1. 确认蓝牙图标在闪烁
2. 将血压计放在 iPhone 旁边（< 10 cm）
3. 重启 iPhone
4. 重新尝试
```

---

## 🔧 故障排查（按顺序）

### 问题：屏幕不亮
```
解决：
1. 检查电池方向（+ - 极性）
2. 更换全新电池
3. 长按电源键 5 秒
```

### 问题：没有蓝牙图标
```
解决：
1. 长按 "MEMORY" 按钮 5 秒
2. 或长按 "SET" 按钮 5 秒
3. 或重启设备
```

### 问题：蓝牙图标常亮（不闪烁）
```
解决：
1. 取出电池 30 秒
2. 重新装入
3. 开机后立即查看 iPhone
```

### 问题：iPhone 看不到设备
```
解决：
1. 确认蓝牙图标在闪烁
2. 缩短距离到 < 10 cm
3. 重启 iPhone 蓝牙
4. 重启血压计
```

---

## ⚠️ 重要提示

### ❌ 不要做：
```
❌ 在 iPhone 设置 > 蓝牙 中点击设备配对
❌ 使用快没电的旧电池
❌ 距离超过 5 米
```

### ✅ 要做：
```
✅ 只确认设备出现在 "其他设备" 列表
✅ 使用全新电池
✅ 保持 < 1 米距离
✅ 让 app 自动连接（不要手动配对）
```

## 🚀 运行 App

### 当 iPhone 能看到设备后：

```
1. 在 Xcode 按 ⌘ + R
2. 等待 2 秒（app 会自动连接）
3. 查看 Xcode 控制台
4. 等待 "🎉 设备已就绪"
5. 在 app 界面看到 "Connected" 🟢
```
