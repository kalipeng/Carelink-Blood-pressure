# 🩺 快速诊断：为什么 iPhone 看不到 KN-550BT

## 🚨 问题：在 iPhone 设置 > 蓝牙 中看不到设备

---

## ⚡ 2 分钟快速检查

### 🔋 检查 1：电池（30 秒）

```
打开血压计背面电池仓，检查：

[ ] 电池已安装
[ ] 电池方向正确（+ - 极性对应）
[ ] 电池有电（建议使用新电池）

如果电池没电 → 更换新电池
```

---

### 💡 检查 2：设备开机（30 秒）

```
按下血压计的 "电源" 或 "START" 按钮

看到：
[ ] 屏幕亮起
[ ] 显示 "0" 或 "--" 或任何内容

如果屏幕不亮 → 更换电池或长按 5 秒
```

---

### 📶 检查 3：蓝牙图标（30 秒）⭐⭐⭐⭐⭐

**这是最关键的！**

```
在血压计屏幕上查找蓝牙图标：

✅ 理想状态：蓝牙图标在闪烁（每秒 2-3 次）
   → 说明设备在配对模式，继续下一步

⚠️ 次要状态：蓝牙图标常亮（不闪烁）
   → 说明蓝牙开启但不在配对模式
   → 需要手动进入配对模式（看下面）

❌ 问题状态：没有蓝牙图标
   → 蓝牙功能未激活
   → 需要手动激活（看下面）
```

---

### 📱 检查 4：iPhone 蓝牙（30 秒）

```
打开 iPhone "设置" > "蓝牙"

[ ] 蓝牙开关是绿色（开启状态）
[ ] 在 "其他设备" 部分等待 10 秒

看到 "KN-550BT" 或 "iHealth" 开头的设备？
   ✅ 是 → 太好了！（但别点它，直接用 app）
   ❌ 否 → 设备没有进入配对模式
```

---

## 🔧 快速解决方案

### 💥 最可能的原因：设备没有进入配对模式

即使设备开机了，蓝牙功能也开启了，**仍然需要手动进入配对模式**才能被 iPhone 发现。

---

## 🎯 3 个最有效的方法（按顺序尝试）

### 方法 1：重启设备（70% 成功率）⭐⭐⭐⭐

```
1. 关闭血压计（长按电源键或取出电池）
2. 等待 10 秒
3. 重新开机
4. 立即打开 iPhone 设置 > 蓝牙
5. 在 10 秒内查看是否出现设备
```

**为什么？** 某些设备只在开机后的短时间内自动进入配对模式。

---

### 方法 2：长按 MEMORY 按钮（85% 成功率）⭐⭐⭐⭐⭐

```
1. 确保设备已开机（屏幕亮着）
2. 找到 "MEMORY" 或 "M" 按钮
3. 长按 5-10 秒
4. 观察屏幕变化：
   • 可能显示 "Pair" 或 "bt"
   • 蓝牙图标开始闪烁
5. 此时查看 iPhone 蓝牙列表
```

**没有 MEMORY 按钮？** 尝试：
- 长按 "SET" 按钮 5 秒
- 长按 "USER" 按钮 5 秒
- 长按电源按钮 5 秒

---

### 方法 3：重置设备（95% 成功率）⭐⭐⭐⭐⭐

```
1. 取出所有电池
2. 等待 30 秒
3. 重新装入电池
4. 按电源开机
5. 设备会自动进入配对模式
```

**或者按 RESET 按钮：**
```
1. 找到设备背面或侧面的小孔（标记 "RESET"）
2. 用牙签或回形针插入
3. 按住 3-5 秒
4. 设备会重置并进入配对模式
```

---

## 🔍 如何确认进入了配对模式

### 在血压计屏幕上：

```
✅ 蓝牙图标快速闪烁（每秒 2-3 次）
✅ 显示 "Pair" 或 "bt" 或 "BLE"
✅ 显示 "---" 或 "rdy"
✅ LED 灯快速闪烁
```

### 在 iPhone 蓝牙设置中：

```
✅ 在 "其他设备" 中出现：
   • "KN-550BT" 或
   • "iHealth KN-550BT" 或
   • "iHealth BP" 或
   • "BP Monitor"
```

## ⚠️ 重要提示

### ‼️ 不要在 iPhone 蓝牙设置中点击设备！

```
❌ 错误做法：
   在 iPhone 设置 > 蓝牙 中点击 "KN-550BT" 进行配对

✅ 正确做法：
   1. 只确认设备出现在列表中
   2. 关闭蓝牙设置
   3. 打开 carelink app
   4. 让 app 自动连接
```

**为什么？** KN-550BT 是 BLE（低功耗蓝牙）设备，使用自定义协议，必须通过专用 app 连接，不能通过系统蓝牙配对。

## 💡 常见问题

### Q: 为什么我的血压计开机后 iPhone 还是看不到？

**A:** 开机 ≠ 配对模式。大多数蓝牙设备需要额外的操作才能进入配对模式（如长按某个按钮）。

---

### Q: 我看到蓝牙图标了，但是常亮不闪烁

**A:** 这说明蓝牙功能开启了，但没有在广播（配对模式）。尝试：
- 长按 MEMORY 按钮 5 秒
- 或重启设备

---

### Q: 我在 iPhone 上看到设备了，但点击后无法配对

**A:** 不要在系统蓝牙中配对！KN-550BT 使用自定义协议，必须通过 carelink app 连接。

---

### Q: 设备之前能用，现在突然看不到了

**A:** 可能是配对状态异常。解决方法：
1. 在 iPhone 设置 > 蓝牙 > "我的设备" 中找到 KN-550BT
2. 点击右边的 (i)
3. 点击 "忽略此设备"
4. 重新进入配对模式
